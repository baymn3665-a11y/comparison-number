# Comparison Number

A beginner-friendly Python project that compares two numbers and determines whether the first number is greater than, less than, or equal to the second.

## Features

- Compare two numbers.
- Supports integer and float values.
- Input validation.
- Exception handling using try, except, and raise.
- Displays whether the first number is greater than, less than, or equal to the second.

## Concepts Used

- Variables
- User Input
- Functions
- if / elif / else
- try / except
- ValueError
